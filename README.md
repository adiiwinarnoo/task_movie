# task_movie
Author Adi Winarno
Using Design Pattern MVVM without coroutine and Dependency Injection
Movie task with API apithemoviedb.org

Feature Technology:
- Retrofit2
- ViewModel LifeCycle
- LifeCycle live data
- Glide
- Constraint Layout
- Webview
- NestedScrollView
